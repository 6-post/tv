## How to start

```
1 › git clone --recursive git@github.com:6-post/tv.git
2 › npm install
3 › enb server
4 › open http://localhost:8080/desktop.bundles/index/index.html

OR

1 › git clone git@github.com:6-post/tv.git
2 › npm install
3 › git submodule init
4 › git submodule update
5 › enb server
6 › open http://localhost:8080/desktop.bundles/index/index.html
```
